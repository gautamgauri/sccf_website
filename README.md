# Suchitra Cancer Care Foundation (SCCF) Website

A nonprofit website focused on cancer awareness, patient guidance, and education for communities in Eastern India.

## About

Suchitra Cancer Care Foundation (SCCF) is a nonprofit initiative founded in loving memory of Late Suchitra Gauri. We focus on clear, responsible, and humane cancer awareness and navigation support for patients, caregivers, and communities.

## Features

- **Cancer Awareness & Education**: Clear, non-technical information about common cancers
- **Patient Guidance & Navigation**: Helping patients understand care pathways
- **Onco Talks**: Educational conversations with experienced oncologists
- **Cancer Sathi**: Upcoming AI-powered cancer awareness assistant (pilot)

## Deployment

This website is deployed via GitHub Pages. The site is automatically published from the `main` branch.

### Local Development

Simply open `index.html` in a web browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

Then visit `http://localhost:8000`
